# 律动之下 UnderGrooves

> 潜入律动之下，听见不一样的回响。

---

## 一、项目概述

**律动之下（UnderGrooves）** 是一个专注于小众音乐推荐与深度介绍的静态网站。站名取 "Under"（地下）与 "Grooves"（律动）的双关含义——挖掘那些不追流量、不被算法裹挟的好音乐，带听众潜入主流视野之外的律动深处。

网站共收录 **15 张专辑**，横跨五大音乐板块，时间跨度从 1967 年到 2024 年，覆盖近 60 年的全球音乐版图。

---

## 二、制作动机

在流量为王的时代，音乐流媒体平台的算法推荐倾向于推送最安全、最大众化的内容。许多优秀但不够"主流"的音乐作品因此被埋没——The Velvet Underground 的先锋实验、具岛直子的都市抒情、King Crimson 的史诗前卫……这些在各自流派中不可替代的经典，在算法世界却鲜有曝光。

**制作动机有三：**

1. **对抗算法茧房**——不以播放量为标准，只以音乐本身的质量与表达为准则，为每一张专辑撰写有温度的深度介绍。
2. **搭建聆听桥梁**——通过视觉（封面）、文字（介绍）、听觉（背景音乐）三重维度，帮助听众建立与音乐的感性连接。
3. **存档个人审美**——将那些陪伴过日夜的好音乐系统整理，既是分享，也是一份个人音乐审美的归档。

### 4.2 视觉设计

- **配色方案**：暗色主题（#080812 底色）+ 半透明深蓝容器，各分类配有独立识别色——电子青（#00d4ff）、摇滚红（#ff6b6b）、嘻哈橙（#ffb347）、华语绿（#6bcb77）、日系粉（#e8a0bf）。
- **字体系统**：优先使用系统原生字体栈（SF Pro Text / PingFang SC / Noto Sans SC），抗锯齿渲染，正文 15px / 行高 1.9。
- **背景处理**：Joy Division《Unknown Pleasures》脉冲星波形图作为全站背景，叠加 90% 黑色遮罩，呼应"地下"主题。
- **布局结构**：1000px 居中容器，Flex 导航 + Grid 卡片网格 + Flex 详情页双栏，响应式适配（≤768px 降级为两栏/单栏）。

### 4.3 技术实现

- **前端**：纯 HTML5 + CSS3，无框架依赖
- **布局**：Div+CSS，`#container` 固定宽度 1000px 居中
- **音频**：HTML5 `<audio>` 元素 + JavaScript 交互解锁（移动端 `touchend` 事件触发，监听 `canplay` 确保加载完毕即播）
- **版本管理**：Git + 远程部署至 GitHub Pages
- **访问链接**：[https://apartus.github.io/undergrooves/](https://apartus.github.io/undergrooves/)

## 五、实用性

1. **音乐发现工具**：15 张专辑涵盖了电子、摇滚、嘻哈、华语独立、日系 City Pop 五大方向，每篇介绍从封面视觉、编曲特点、声线特质到适合场景都有详细分析，读者可据此判断是否合自己口味。
2. **跨平台访问**：部署于 GitHub Pages，任何设备通过浏览器即可访问，无需安装任何应用。桌面端和移动端均已适配。
3. **沉浸式体验**：每个分类页面配有对应风格的背景音乐，进入页面的瞬间即可感受该流派的氛围基调。
4. **可维护性**：纯静态 HTML，内容与样式分离（CSS 外部引用），新增专辑只需添加 HTML 卡片 + 图片 + 介绍文本，无需改动核心架构。
5. **学术参考**：作为 Web 前端课程的期末项目，完整展示了静态网站从设计到部署的全流程，涵盖了 HTML 语义化、CSS 布局、响应式设计、多媒体处理、用户体验等核心知识点。

---

## 六、总结

**律动之下** 从最初的 12 张专辑扩展到 15 张，从 6 个页面扩展到 7 个，历经多轮迭代优化——从基础布局搭建、流派配色体系建立，到移动端音频兼容性调试、MP3 压缩与封面图剥离，再到字体排版精调、交互细节打磨——每一轮修改都在追求更好的用户体验。

这个项目不只是一个课程作业。它试图回答一个问题：**在一个算法支配听觉的时代，人应该如何主动地、有态度地发现好音乐？** 答案藏在这 15 张专辑的封面、文字与旋律里——潜入律动之下，听见算法之外的频率。
